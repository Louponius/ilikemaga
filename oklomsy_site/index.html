How does the server sort traffic to different websites hosted on the same server?

Well first off, we check the url to see if the user is requesting a specific site and we redirect them there. this happens if you go to the root of the server.

we can add different domains by simply changing the JavaScript file inside of the root and the DNS of said domain.

we add a "A" DNS record that points to this server's IP with a subdomain or none at all.

We can't use CNAME records because they do not allow you to put a IP. they only allow domain names so unless you have a domain name that points to a file that redirects you to the IP you want, it's better to use A records instead.

There is a big problem with "A" records though, they redirect your domain to the ENTIRE server, so through this site I can access other sites stored on this server that I shouldn't be able to, but that's okay. we don't store anything secret in here, well, we used to.
